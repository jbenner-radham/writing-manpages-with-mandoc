Quotes and Text Enclosures
==========================
Quotations and enclosure symbols for text.

`Aq`
----
Enclose in angle brackets.

### Markup
```mdoc
.Aq 123
```

### Output
> &lang;123&rang;

`Bq`
----
Enclose in square brackets.

### Markup
```mdoc
.Bq Hello, world.
```

### Output
> [Hello, world.]

`Brq`
-----
Enclose in curly braces.

### Markup
```mdoc
.Brq Hello, world.
```

### Output
> {Hello, world.}

`Dq`
----
Enclose in typographic double quotes.

### Markup
```mdoc
.Dq Hello, world.
```

### Output
> ``Hello, world.''

`Pq`
----
Enclose in parentheses.

### Markup
```mdoc
.Pq Hello, world.
```

### Output
> (Hello, world.)

`Qq`
----
Enclose in &ldquo;typewriter&rdquo; double quotes.

### Markup
```mdoc
.Qq Hello, world.
```

### Output
> "Hello, world."

`Sq`
----
Enclose in single quotes.

### Markup
```mdoc
.Sq Hello, world.
```

### Output
> `Hello, world.'
